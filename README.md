# 🛡️ Blue Sentinel SOC Lab

## Overview

Welcome to my Blue Team Home Lab.

This project documents my journey toward becoming a Security Operations Center (SOC) Analyst through hands-on learning, practical labs, and professional documentation.

---

## Project Goals

- Build an enterprise-style SOC environment
- Learn Linux and Windows administration
- Deploy Wazuh SIEM
- Configure Active Directory
- Implement endpoint monitoring with Sysmon
- Simulate attacks safely
- Investigate incidents
- Practice threat hunting
- Map detections to MITRE ATT&CK

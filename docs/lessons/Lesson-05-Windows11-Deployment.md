# Lesson 05 – Windows 11 Deployment

## Objective

Deploy the Windows 11 endpoint (WIN11-01) for the Blue Sentinel SOC Lab.

## VM Specifications

- Name: WIN11-01
- OS: Windows 11 Enterprise Evaluation
- RAM: 4 GB
- CPU: 2 vCPUs
- Disk: 60 GB (Dynamic)

## Network

- NAT

## Configuration

- Local account created
- Computer renamed
- Guest Additions installed
- Windows Update completed

## Validation

- Windows boots successfully
- Login successful
- Internet connectivity verified
- Snapshot created
